Getting started
---------------

There are many aspects to the TEMUL Toolkit, such as polarisation analysis,
element quantification, and automatic image simulation (through pyprismatic).

Checkout the tutorials in the table of contents above or on the left of the page.
One can also view the extensive :ref:`documentation <api_doc>`, where each function
is described and examples of their use given.
